# Cyber Battle Animation

Animated cyber battle scene built with React, TypeScript, and Vite.

## Run locally

1. Install dependencies:

   npm install

2. Start the dev server:

   npm run dev

3. Build for production:

   npm run build

## Why this is not animated in GitHub mobile app

The GitHub app shows repository files and markdown, but it does not run this React app runtime.

To view the animation, use one of these:

- Local dev server URL from npm run dev
- A deployed URL (for example GitHub Pages)
